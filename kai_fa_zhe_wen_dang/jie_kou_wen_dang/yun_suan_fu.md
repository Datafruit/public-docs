# 运算符

| **名称** | **描述** |
| --- | --- |
| + | 加法运算符 |
| - | 减号运算符 / 一元求反 |
| \* | 乘法运算符 |
| / | 除法运算符 |
| =, IS | 等于运算符 |
| !=, &lt;&gt;, IS NOT | 不等于运算符 |
| &gt;= | 大于或等于运算符 |
| &gt; | 大于运算符 |
| &lt;= | 小于或等于运算符 |
| &lt; | 小于运算符 |
| BETWEEN ... AND ... | 检查一个值是否在值范围内 |
| NOT BETWEEN ... AND ... | 检查一个值是否不在值范围内 |
| LIKE, CONTAINS | 简单的模式\(模糊\)匹配 |
| NOT LIKE, CONTAINS | 否定的简单模式\(模糊\)匹配 |
| REGEXP | 模式匹配使用正则表达式 |
| NOT REGEXP | 正则表达式的否定 |
| NOT, ! | 取否 |
| AND | 逻辑并 |
| OR | 逻辑或 |
